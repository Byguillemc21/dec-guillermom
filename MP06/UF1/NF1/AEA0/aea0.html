1. Què és Vanilla Javascript?

Vanilla JavaScript es javascript en su version mas pura ,es decir sin tener ninguna libreria o framework que te pueda facilitar a la hora de programar.

2. Diferència els ECMA Script que pertanyen a Javascript i a Javascript Modern.

ECMA Script es la especificacio on es mencionen tots els detalls de com te que funcionar i comportar javascript en un navegador i el modern s'encarrega.

5. Què és una arrow function? A quin ES apareix (quin any).

Un arrow function es una forma de definir funcions de manera mes curta(no cal ficar parentesis ni claus), tambe te distintes variants en la sintaxi.

6. Què és un callback? Posa un exemple.

Un callback son les funcions que son asignades a un parametre a una altre funcio.
Un exemple =>

function saludar(nombre) {
      alert('Hola ' + nombre);
}

function procesarEntradaUsuario(callback) {
      var nombre = prompt('Por favor ingresa tu nombre.');
      callback(nombre);
}

procesarEntradaUsuario(saludar);

7. Què és una promesa en Javascript (promise). Posa un exemple. Explica també la diferència entre la programació síncrona i la programació asíncrona, posa un exemple.

Una promesa en JavaScript es un objecte que representa la terminacio o el fracas d'una operacio asincrona, per exemple un metode de promese es el .then(funcio) -> aquest es la forma general d'utilitzar una promesa.
La programació síncrona es tot allo que te que esperar fins que el resultat d'un codi s'hagi fet mentres que la programació asíncrona es tot allo que dona una resposta que es donara en un futur sense que es tingui que esperar a la resposta.
Un exemple de síncrona => console.log("hola");
Un exemple de asíncrona =>

function dos{} {
      console.log("Dos bro");
};

functio uno() {
      console.log("Uno bro");
      dos();
      console.log("Tres bro");
};

uno();
